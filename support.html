Frequently Asked Questions

What's a GoDex?

GoDex is an online Pokedex checklist for Pokemon GO. Use it to keep track of your Pokemon collection outside of the game.

How does it work?

Just tick off all the Pokemon you have so that you can see all the ones you still need. All of your changes are saved automatically, neat huh.

What do the different menu buttons do?

- A page for keeping track of your 100% Pokemon.

- A page for keeping track of your Pokemon of different gender.

- A page for keeping track of your lucky Pokemon.

- Hides the Pokemon that you ticked off so that you only see the ones you still need.

- Shows you Pokedex completion percentages of all the pages.

- Generates a unique code for your collectiom which allows you to save it or maybe import this code on another device.
